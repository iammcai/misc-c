# ARP

基于已实现的zcap和ftx，我们可以实现ARP探测、扫描功能

## ARP探测

探测某个IP的MAC地址，通过发送ARP请求来实现

我们可以在zcap中注册ARP条件，将ARP报文都通过注册的回调函数处理

全局维护一个ARP reply哈希表，当收到ARP reply时，更新对应的条目

- 哈希表中无条目，创建并加入，设置老化定时器
- 哈希表中有条目，若reply结果不一致，进行更新。且需要重新设置老化定时器

probe的流程为：

1. 发送ARP探测报文，等待一段时间
2. 期间若收到ARP响应，会更新到Cache中
3. 等待结束时，从Cache中获取结果
