# MISC-C

## 写在前面

本项目用于个人学习，提高C编码能力。目的是实现一套可用高性能轮子

项目同步更新在个人公众号[codeFishh]，欢迎star

## 施工进度

- 数据结构
    - 侵入式链表，性能优于非侵入式链表
    - 侵入式哈希表，性能优于非侵入式哈希
    - 侵入式原子SPSC队列，性能优于mutex+队列
    - 原子MPMC队列，解决ABA问题，TODO：解决高并发时的use-after-free问题
    - 侵入式堆，用于后续优先队列使用
- 组件
    - 事件驱动线程框架，功能ok
    - 用户态锁框架，支持自动加锁解锁，目前实现自旋锁、读写锁，性能优于pthread_mutex，实现安全的mutex宏
    - 内存池，支持跨线程收发，功能ok，性能优于calloc/free
    - 消息队列，支持多发一收
- 管理组件
    - 一个简易的CLI服务，使用前缀/字典树来存储命令
